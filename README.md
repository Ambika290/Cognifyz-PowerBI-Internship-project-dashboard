# Cognifyz Power BI Internship Project

## Overview

This project was completed as part of the "Power BI Internship" at "Cognifyz Technologies".

The objective of this internship was to analyze an investment dataset using Power BI and create interactive dashboards that provide meaningful business insights. The project includes data visualization, KPI cards, DAX measures, slicers, matrices, and interactive reports.

---

# Tools Used

- Microsoft Power BI Desktop
- Microsoft Excel
- Power Query
- DAX (Data Analysis Expressions)

---

# Project Tasks

## Task 1 – Data Exploration & Summary

### Objective

Analyze the overall investment dataset and create summary visualizations.

### Dashboard Includes

- Average Age KPI Card
- Total Respondents KPI Card
- Percentage of Investors KPI Card
- Gender Distribution (Donut Chart)
- Savings Purpose Analysis
- Investment Avenue Distribution
- Average Investment Ratings
- Business Insights

# Key Insights

- Most respondents have invested in financial products.
- Wealth Creation is the most common investment purpose.
- Capital Appreciation is the leading investment objective.
- Investment preferences vary across gender.
- Financial Consultants are a major source of investment information.
- Most investors prefer medium-term investments.
- Different investment avenues are chosen for different reasons.
